Steps in this series:
Strict mode:
  1. You will be presented with a random series of button presses to replicate, starting with one.
  2. Each time you input a series of button presses correctly, you will see the same series of button presses but with an additional step.
  3. When in strict mode, if you get a button press wrong, it notifies you that you have done so, and the game restarts at a new random series of button presses.

  4. To turn strict mode OFF and ON, use the selector once the game has started.

  5. When not in strict mode, if you press the wrong button, you will be notified that you have done so, and that series of button presses will start again to remind you of the pattern so you can try again.
  6. You can see how many steps are in the current series of button presses.
  7. If you want to restart, you can hit the reset button and the game will return to a single step.
  8. You can win the game by getting a series of 20 steps correct.
